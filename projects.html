Projects

Distributed Wastewater Surveillence Platform Leveraging Hybrid Microfluidic CMOS Biosensors

This project uses microfluidics to interface between luminescent synthetic microbial biosensors and low power photosensing electronics, providing a real time monitoring method for harmful substances in wastewater.

The Automated Plug & Playable Microfluidic System for Cell Communication

This project introduces a modular microfluidic system using 3D-printed components and an Arduino-controlled pump mechanism. It allows for precise liquid manipulation in low-resource settings.

GLM Boost for Immune Response Prediction

GLM Boost was used to predict immune response to vaccines from various clinical features.

CpG Island Classifier

A Python-based classification tool using Markov chains to identify CpG islands in genomic DNA, helping to predict regulatory regions involved in gene expression and disease risk.

Bayesian Networking for Breast Cancer Survival Predictability

A Python-based bayesian inference tool used clinical gene mutation data to predict breast cancer survivability.

Lateral Flow Assay for Drug-Facilitated Sexual Assault Prevention

We designed and prototyped a portable test for gamma hydroxybutyric acid (GHB) in beverages, which is one of the most common drugs used in drug-facilitated sexual assault. The mechanism leveraged bifuncitonalized silver nanoparticles and gave a colorimetric result.
